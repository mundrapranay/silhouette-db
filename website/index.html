Cutting-Edge Technology

Built with the most advanced cryptographic primitives and distributed systems techniques

Oblivious Key-Value Store (OKVS)

Random Band Matrix Encoding

OKVS encodes key-value pairs into a single compact blob that reveals nothing about which keys were stored. Any key can be decoded from the blob, providing complete storage obliviousness.

Encoding Overhead 10-20%
Key Hash BLAKE2b512
Min Pairs 100

Private Information Retrieval (PIR)

FrodoPIR with LWE Cryptography

FrodoPIR enables clients to query the database without revealing which key they're requesting. The server processes every element but learns nothing about the queried index.

Security Post-Quantum
LWE Dimension 512-1572
Latency < 10ms

Quick Start 
# Clone the repository
git clone https://github.com/mundrapranay/silhouette-db.git
cd silhouette-db

# Initialize submodules and build
make submodule-init
make build

# Start the server with OKVS backend
./bin/silhouette-server \
    -node-id=node1 \
    -listen-addr=127.0.0.1:8080 \
    -grpc-addr=127.0.0.1:9090 \
    -data-dir=./data/node1 \
    -bootstrap \
    -storage-backend=okvs
